# VS Code Workspace Settings for Ghost Theme Development

These settings are automatically applied when working in the devcontainer.

## Recommended Extensions

The devcontainer automatically installs:
- ESLint - Code linting
- Prettier - Code formatting  
- Tailwind CSS IntelliSense - CSS utility class completion
- Ghost - Ghost theme development support

## Editor Settings

- Auto-format on save enabled
- Prettier as default formatter
- Consistent indentation and line endings
